Music store analysis SQL

Project Objective:
The objective of this project is to analyze a digital music store database to extract meaningful business insights such as customer behavior, sales trends, and product performance.
The analysis helps stakeholders understand revenue drivers, top customers, and popular genres to support data-driven decision making.

Dataset Description:
Artist
Album
Track
Playlist
Media type
Genre
Customer information
Invoice
InvoiceLine
Employee

SQL Concepts Covered:
SELECT, WHERE, ORDER BY
JOINS (INNER, LEFT, RIGHT)
GROUP BY & HAVING
Aggregate Functions (SUM, COUNT, AVG)
Subqueries
Common Table Expressions (CTE)
